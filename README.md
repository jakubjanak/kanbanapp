# Kanban App

Tuto aplikaci jsem se rozhodl vytvořit kvůli pracovním projektům. Notion mi přijde pro mé osobní projekty overkill.

## Tech stack
- React
- Typescript
- Tailwind
- Vite

## Instalace

1. Naklonování repozitáře
```bash
git clone https://github.com/jakubjanak/kanbanapp.git
```
2. Přesunutí se do složky projektu
```bash
cd kanbanapp
```
3. Instalace
```bash
npm install
```
4. Spuštění
```bash
npm run dev
```
